# rustydump

Rustydump is a modern, objdump-compatible binary analysis tool written in Rust for i386 and amd64.
It supports disassembly, symbol inspection, section/header parsing, and control-flow graph generation [EXPERIMENTAL] for
multiple object file formats.
It features selectable disassembly backends (iced-x86 and Zydis) and optional IDA-style enhanced output including labels
and cross-references.
